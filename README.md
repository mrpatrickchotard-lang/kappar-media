# Kappar Media

A modern media platform for business leaders.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content Structure

- `content/articles/` - Markdown articles
- `content/authors/` - Author profiles
- `public/images/` - Static images
